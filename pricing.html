МАТЕРІАЛ

Чим друкує 3D-принтер?

Витратні матеріали (філаменти) для 3D-принтерів являють собою пластикові нитки, намотані на котушки. Витратні матеріали бувають різних типів і властивостей. Про всі типи матеріалів ви можете прочитати в енциклопедії 3Dtoday.
Кілограм самого дешевого пластику можна купити за якісь 250 грн. Хоча більш цікаві варіанти (наприклад, імітатори деревини або пісковика з наповнювачами зі справжньої деревини або каменю) вже можуть обійтися в кілька разів дорожче.

Найпопулярніших матеріалів для 3D друку

PLA

PLA пластик - безумовний король в умовах домашнього друку. Його чато порівнюють з ABS - другий за популярністю - але все одно PLA значно обганяє ABS. І на те є причини.Перша і найважливіша PLA пластиком легко друкувати

  • Міцність: висока | гнучкість: низька | довговічність: середня
  • Складність використання: Низька
  • Температура друку: 180 ° C - 230 ° C
  • Температура столу для друку: 20 ° C - 60 ° C (не обов'язково)
  • Усадка / деформації при охолодженні: мінімальна
  • Розчинний: Ні
  • Екологічно безпечний: Залежить від виробника

ABS

За своїми властивостями ABS є своєрідним поліпшенням PLA, хоча друкувати їм набагато складніше.При цьому, ABS пластики активно використовуються у виробництві.З цього матеріалу виготовляють, наприклад, цеглинки LEGO і шоломи для мотоциклів

  • Міцність: висока | гнучкість: середня | довговічність: висока
  • Складність використання: середня
  • Температура друку: 210 ° C - 250 ° C
  • Температура столу для друку: 80 ° C - 110 ° C
  • Усадка / деформації при охолодженні: терпима
  • Розчинник: Ацетон і його аналоги
  • Екологічно безпечний: немає

PETG, PET, PETT

Polyethylene terephthalate - найпоширеніший вид пластика в світі.Вам він може бути знаком як полімер, який використовується для пляшок з-під води, контейнерів для їжі. Його різновид - PETG - досить поширений матеріал.

  • Міцність: висока | гнучкість: середня | довговічність: висока
  • Складність використання: низька
  • Температура друку: 220 ° C - 250 ° C
  • Температура столу для друку: 50 ° C - 75 ° C
  • Усадка / деформація: мінімальна
  • Розчинна: немає
  • Екологічно безпечний: Залежить від виробника
